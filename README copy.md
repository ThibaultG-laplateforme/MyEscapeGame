# Escape Game Virtuel en 3D

### Description

Ce projet vise à créer un **escape game virtuel immersif** en 3D temps réel, accessible en ligne pour plusieurs joueurs. Les participants doivent résoudre des énigmes collaboratives dans un environnement 3D détaillé, avec une gestion du temps et un scénario captivant. Le jeu peut être joué en solo ou avec un groupe (2 à 6 joueurs) à distance, via un navigateur ou un client léger sur PC, avec option de réalité virtuelle.

### Objectifs principals

- **Environnement 3D immersif** : Décors détaillés et interactifs, déplacements fluides des avatars, ambiance sonore et visuelle.
- **Énigmes et puzzles variés** : Logique, manipulation d'objets, cryptogrammes, avec des indices visuels et auditifs.
- **Collaboration en ligne** : Chat vocal et textuel intégré, énigmes nécessitant une coopération entre joueurs, rôles spécifiques.
- **Gestion du temps** : Sessions limitées à 60-90 minutes avec compteur visible.
- **Scénario évolutif** : Histoire immersive débloquant de nouvelles pièces ou révélant des éléments narratifs.
- **Multijoueur** : Synchronisation en temps réel des actions des joueurs, accessible sur PC avec une option VR.
- **Sécurité** : Protection des données personnelles (RGPD), système de comptes sécurisés et modération des interactions.

### Technologies

- **Moteur de jeu** : [3DVerse](https://www.3dverse.com) pour la création d'environnements 3D.
- **Multijoueur** : Système de synchronisation en temps réel pour les actions des joueurs.
- **Interface utilisateur (UI)** : HUD pour la gestion du temps, des indices et de la communication.

### Prérequis

- **Systèmes d'exploitation** : Windows, Mac
- **Navigateur** (si joué via navigateur) ou **client léger** (si installé).
- **Connexion internet stable** pour une expérience multijoueur fluide.
- **Option VR** : Casque VR facultatif pour une expérience encore plus immersive.

### Installation

1. **Téléchargement** :
   - jouez directement via votre navigateur à l'adresse [lien-du-jeu](#).

2. **Configuration** :
   - Pour la version client, installez le fichier téléchargé en suivant les instructions.
   - Vérifiez que vous disposez d'une connexion internet stable.

3. **Lancement** :
   - Démarrez le jeu en lançant l'application ou en vous connectant sur le site web.

### Utilisation

- Créez une session multijoueur ou rejoignez une session existante.
- Utilisez les commandes du HUD pour interagir avec l'environnement.
- Résolvez les énigmes avec vos coéquipiers pour progresser dans l'histoire.

### Sécurité et protection des données

- Ce jeu respecte les normes de protection des données personnelles (RGPD).
- Les joueurs doivent créer un compte avec un email et un mot de passe sécurisé.
- Un système de modération assure la qualité des interactions entre les joueurs.
